# Currency Converter

A Currency Converter made using React, TypeScript and ExchangeRateAPI. It has features like converting currencies from one currency to another, swapping currencies, and displaying the exchange rate.

# Tech Stack
- React
- TypeScript
- Tailwind CSS
- ExchangeRateAPI

# How to Run
1. Clone the repository with `git clone <repository_url>`
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
